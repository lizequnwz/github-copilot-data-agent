# Generated analytical artifacts

`reports/generated/` contains local artifacts produced during exploration or after validation:

- Markdown analysis records and editable Jupyter notebooks.
- Python-rendered accessible SVG bar, line, or additive waterfall charts, including multi-series
  bar and line comparisons.
- Self-contained responsive HTML reports with structured insights, one to three focused charts,
  keyboard/touch tooltips, series toggles, a sortable/filterable complete table, definitions,
  methodology, notes, and an optional SQL appendix.

Exploratory reports are visibly labeled when result checks have not run. HTML reports include
light/dark and print styles and load no remote scripts, fonts, or assets. Generated output is
ignored by Git.
